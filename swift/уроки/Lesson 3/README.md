<h1>Урок 3: Циклы, случайные числа, специальные операторы и работа с UIPickerView</h1>

<h2>Циклы в Swift</h2>
<p>Циклы предназначены для многократного выполнения блоков кода.</p>

<h3>Цикл for-in:</h3>
<p>Проходит по диапазону чисел или элементов массива.</p>
<br/>
<p>Пример:</p>

```swift
for number in 1...5 {
    print(number)
}
```

<br/>
<h3>Цикл while:</h3>
<p>Продолжает выполнение, пока выполняется указанное условие.</p>
<br/>
<p>Пример:</p>

```swift
var counter = 0

while counter < 5 {
    print(counter)
    counter += 1
}
```

<br/>
<h3>Цикл repeat-while:</h3>
<p>Сначала выполняет тело цикла, затем проверяет условие.</p>
<br/>
<p>Пример:</p>

```swift
var count = 0

repeat {
    print(count)
    count += 1
} while count < 5
```

<br/>

<h4>Специальные операторы в Swift</h4>
<br/>
<h5>Диапазоны (1...3, 1..<3)</h5>
<p>Диапазоны позволяют указывать интервалы чисел:</p>
<ul>
  <li><b>1...3</b> - закрытый диапазон (включает обе границы: 1, 2, 3)</li>
  <li><b>1..<3</b> - полуоткрытый диапазон (включает начало, но не конец: 1, 2)</li>
</ul>

<br/>
<p>Пример:</p>

```swift
for i in 1...3 {
    print(i) // выведет 1, 2, 3
}

for j in 1..<3 {
    print(j) // выведет 1, 2
}
```

<br/>

<h5>Оператор объединения с нулевым значением (??)</h5>
<p>Этот оператор используется для замены nil (отсутствующее значение) на другое значение по умолчанию:</p>
<br/>
<p>Пример:</p>

```swift
let optionalValue: String? = nil
let safeValue = optionalValue ?? "Значение по умолчанию"

print(safeValue) // выведет "Значение по умолчанию"
```

<br/>

<h2>Работа с опционалами и специальными символами (?, !)</h2>
<p>В Swift многие значения могут быть опциональными, то есть потенциально отсутствующими. Для работы с такими значениями используются специальные символы:</p>
<ul>
  <li><b>?</b> -обозначает опционал (возможно nil)</li>
  <li><b>!</b> - принудительное развертывание опционала (предполагается, что значение не nil)</li>
</ul>

<br/>
<p>Пример:</p>

```swift
let array: [String]? = ["один", "два", "три"]
let element = array?[0] // Опциональный доступ к элементу

let forcedElement = array![0] // Принудительное развертывание (может вызвать ошибку, если array == nil)
```

<br/>

<h2>Генерация</h2>
<p><b></b>Генерация</b> — это когда компьютер сам создаёт что-то: числа, слова, картинки и т. д. Например, в игре компьютер может сам выбрать, куда поставить врага, или в приложении — сгенерировать случайный вопрос для викторины.</p>
<br/>

<h3>Генерация чисел</h3>
<br/>

```swift
let randomNumber = Int.random(in: 1...10)
print("Случайное число: \(randomNumber)")
```

<br/>
<p>Где:</p>
<ul>
  <li><b>Int.random(in: 1...10)</b> - команда, которая выбирает случайное целое число от 1 до 10</li>
  <li><b>1...10</b> - диапазон от 1 до 10 включительно</li>
  <li><b>let randomNumber</b> - мы сохраняем выбранное число в переменную, чтобы потом использовать</li>
</ul>
<br/>

<h3>Генерация элементов из коллекции</h3>
<p><b>Коллекция</b> — это «контейнер» для хранения данных.</p>
<p>Самый простой пример — <b>массив</b> (список элементов).</p>

<br/>
<p>Например, у нас есть список животных:</p>

```swift
let animals = ["кошка", "собака", "слон", "тигр"]
```

<br/>
<p>Чтобы выбрать случайное животное, нужно:</p>
<ol>
  <li>Сгенерировать случайный <b>индекс</b> (номер элемента в списке)</li>
  <li>Взять элемент по этому индексу</li>
</ol>
<br/>

```swift
// 1. Генерируем случайный индекс (от 0 до количества элементов - 1)
let randomIndex = Int.random(in: 0..<animals.count)

// 2. Выбираем элемент по этому индексу
let randomAnimal = animals[randomIndex]

print("Случайное животное: \(randomAnimal)")
```

<br/>

<p>В Swift есть удобный метод <mark>randomElement()</mark>, который позволяет выбрать случайный элемент из коллекции (например, массива) <b>без необходимости вручную генерировать индекс</b>. Это упрощает код и снижает риск ошибок.</p>

<br/>
<h4>randomElement</h4>
<p>Порядок работы:</p>
<ol>
  <li>Вызов метода:</li>
  <br/>

```swift
let randomAnimal = animals.randomElement()
```
  
  <br/>
  <li>Метод возвращает <b>опциональное значение</b> (Optional), потому что коллекция может быть пустой — тогда результат будет <mark>nil</mark></li>
  <li>Чтобы безопасно использовать результат, проверяем его:</li>
  <br/>

```swift
if let chosenAnimal = animals.randomElement() {
    print("Случайное животное: \(chosenAnimal)")
} else {
    print("В списке нет животных!")
}
```
  
  <br/>
</ol>
